## Features
- Network Health Prediction
- Radio Link Degradation Detection
- Root Cause Analysis

- Health Score Gauge
- Forecasting Dashboard
- Feature Importance Analysis
- Telecom Analytics Dashboard

- Python
- Streamlit
- Pandas
- NumPy
- XGBoost
- Scikit-Learn
- Plotly

pip install -r requirements.txt
